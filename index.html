PLSAR

Kotlin Framework

PLSAR Kotlin based web framework provides a built in server, handles dependency injection, http routing, data persistence and view data binding.

This allows for rapid development saving you time & money. PLSAR allows you to spend more of your energy on features instead of configuration.

Main.kt

            
import plsar.PLSAR

fun main(){
    val plsar = PLSAR.Builder()
                .port(8080)
                .ambiance(20)
                .create()
    plsar.start()
}
