PLSAR

eco-system

PLSAR Kotlin based web framework has a built in server which handles dependency injection, http routing, data persistence and view data binding or templating

Main.kt

            
import plsar.PLSAR

fun main(){
    val plsar = PLSAR.Builder()
                .port(8080)
                .executors(20)
                .create()
    plsar.start()
}
